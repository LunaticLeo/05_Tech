Java安装
JRE只是运行用，无法开发。JDK8和JDK17都是长期维护的JDK版本，建议JDK17，JDK8支持时间最长，但是Spring Framework 6 将采用 Java 17 和 Jakarta EE 9。
JDK8 https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
JDK17 https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
下载安装包之后需要配置环境，Windows环境下需要添加环境变量，有两种方法
#以管理员权限打开命令行，运行如下命令
setx -m JAVA_HOME "C:\Program Files\Java\jdk-17"
setx -m PATH "%PATH%;%JAVA_HOME%\bin;%JAVA_HOME%\lib;"
#手动打开系统环境
JAVA_HOME: C:\Program Files\Java\jdk-17 #新建
PATH： ;%JAVA_HOME%\bin;%JAVA_HOME%\lib; #原内容上添加
#安装完成后运行cmd运行如下命令，查看是否配置成功，两条命令需都能显示。环境配置需保存之后重开cmd才能生效
java -version
javac -version
